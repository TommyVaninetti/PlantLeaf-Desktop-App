# PLANTLEAF (1.0.0 beta) made by Tommaso Vaninetti

PlantLeaf is a powerful set of tools specifically designed to bridge the gap between rigorous science and creative exploration, 
ensuring scientific accuracy while making all the setup – hardware and software – intuitive to use.


Action Potential
PlantLeaf combines a very precise hardware with a user-friendly python application for both acquisition and analysis, 
empowering the user to detect even the weakest reactions.


Ultrasonic Clicks
Another interesting part is about ultrasonic clicks detection. 
PlantLeaf uses a powerful yet cheap instrumentation to detect even the shortest clicks, 
with the ability to acquire for long hours and replay every experiment to analyse every single detail.


PlantLeaf is designed to allow everyone to study phenomena that were previously the privilege of complex setups and specialized equipment.

Notes: somme functionalities such as Noise Cancellation and Connectivity Check haven't been implemented yet.
